# DeepSpotify

Deep learns your favorite Spotify playlists to generate music.

## An Abbreviated Usage Section

```{bash}
$ python src/main.py -h
Using TensorFlow backend.
usage: main.py [-h] run_opt uri

positional arguments:
  run_opt     An integer: 1 to analyze audio, 2 to train, 3 to generate, 4 to
              do everything.
  uri         A string: spotify playlist uri

optional arguments:
  -h, --help  show this help message and exit
  
```
